# WalkIn

WalkIn is a sleek and modern Android app designed to streamline the walk-in experience for users. Built with a focus on smooth navigation, beautiful UI design, and efficient data handling, the app delivers a polished and responsive experience across all Android devices.

## Features

- **Responsive UI:** Fully optimized for all screen sizes and orientations, including landscape mode, ensuring a seamless experience on every device.
- **Custom Bottom Navigation Bar:** A fully custom-drawn bottom navigation bar with a floating action button (FAB) that animates smoothly across tabs — built from scratch using Canvas drawing, ValueAnimator, and DecelerateInterpolator.
- **Animated Fragment Transitions:** Smooth and fluid animations between fragments for a native, app-like feel during navigation.
- **Navigation Component:** Uses Jetpack Navigation Component for a clean, scalable, and back-stack-aware navigation structure.
- **RecyclerView:** Dynamic lists rendered efficiently using RecyclerView with custom adapters for flexible data display.
- **Image Loading with Glide:** Glide handles all image loading and caching for fast and smooth image rendering throughout the app.
- **Custom Adapters:** Tailored RecyclerView adapters for optimized and dynamic data management.
- **Material Design Components:** Leverages Material Design components for consistent and modern UI elements including cards, inputs, and typography.
- **MotionLayout / Transition Animations:** Smooth UI transitions powered by MotionLayout and Android's transition framework for a premium feel.

## Tech Stack

- **Language:** Kotlin
- **Navigation:** Jetpack Navigation Component
- **UI:** Custom Views (Canvas), MotionLayout, Material Components
- **Image Loading:** Glide
- **Lists:** RecyclerView + Custom Adapters
- **Animations:** ValueAnimator, DecelerateInterpolator, Fragment Transitions
